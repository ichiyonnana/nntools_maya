# Overview
Maya 向けモデリング補助ツールです｡

# Installation
install.bat を実行してください。
実行には PyMEL が必要です｡ PyMEL を簡単にインストールするスクリプトを用意してあります→ [pymel_installer](https://github.com/ichiyonnana/pymel_installer)

PyMEL はいずれ不要にする予定です。

# Basic Usage
以下のコードでランチャーを起動できます｡
```python
import nnlauncher.core
nnlauncher.core.main()
```

もう少し詳しい使用方法は Wiki にあります｡
[Wiki](https://github.com/ichiyonnana/nntools_maya/wiki)
